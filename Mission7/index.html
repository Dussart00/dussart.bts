QCM Réseaux

1. Quel est le protocole utilisé pour la communication sur le web ?

  • HTTP
  • FTP
  • SSH
  • DNS

2. Qu'est-ce qu'une adresse IP ?

  • Un protocole de communication
  • Une adresse physique d'un appareil
  • Une adresse unique d'un appareil sur un réseau
  • Un logiciel permettant de se connecter à Internet

3. Que signifie le terme DNS ?

  • Domain Name System
  • Data Network Service
  • Digital Network Security
  • Domain Network System

4. Quelle est la principale fonction d'un routeur dans un réseau ?

  • Envoyer des e-mails
  • Relier différents réseaux entre eux
  • Connecter un ordinateur à Internet
  • Gérer les connexions sans fil

5. Quelle est la différence entre IPv4 et IPv6 ?

  • IPv4 a une longueur d'adresse plus grande
  • IPv6 permet des adresses plus petites
  • IPv6 a une longueur d'adresse plus grande
  • Aucune différence entre IPv4 et IPv6

6. Quel est le rôle du protocole DHCP ?

  • Assigner une adresse IP automatiquement
  • Coder les données pour sécuriser les communications
  • Créer des sous-réseaux
  • Filtrer les paquets de données sur un réseau

7. Qu'est-ce qu'un switch dans un réseau local ?

  • Un appareil qui connecte des réseaux distants
  • Un appareil qui connecte des ordinateurs sur un même réseau
  • Un appareil qui chiffre les données
  • Un appareil qui remplace un modem

8. Le protocole TCP est utilisé pour ?

  • Transmettre des paquets sans se soucier de l'ordre
  • Assurer une transmission fiable et ordonnée des données
  • Gérer les adresses IP
  • Gérer la résolution des noms de domaine

9. Qu'est-ce qu'un VPN ?

  • Un protocole de sécurité pour la transmission des paquets
  • Un service permettant de masquer l'adresse IP d'un utilisateur
  • Un type de câble pour les réseaux locaux
  • Un réseau social sécurisé

10. Quelle est la fonction du protocole HTTPS ?

  • Assurer la sécurité des communications via une connexion sécurisée
  • Assurer la rapidité de la connexion
  • Gérer l'attribution des adresses IP
  • Assurer une transmission fiable des paquets
